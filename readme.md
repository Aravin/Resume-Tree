## High-Level Features

* Add Resume
* Edit Resume
* Generate Public Resume Link
* Download Resume as PDF
* Download Resume as HTML
* Download Resume as Word

## Low-Level Features

* Account Management
  * Login
  * Register
  * Forget Password
  * Account Details
  * Account Edit

* Resume
  * Create Resume
  * Edit Resume
  * Preview Resume
  * Download Resume
  * Resume Dashboard/Overview

* API
  * Authentication
  * Store Data in Database
  * Generate PDF
  * Generate Image for thumbnail
  * Storage for PDF and Images

## Tech Stacks

* Front-end
  * Next.js (A React.js Framework)
  * TailwindCSS
  * DaisyUI

* Back-end
  * Express.js
  * Puppeteer (For generating image and PDF)
